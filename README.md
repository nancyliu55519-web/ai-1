# 富甲天下 · AI多体系术数问答

> 站点品牌名「富甲天下」（商标）；「问卜」在文案中仅作动词（起局问卜）使用。

## 设计风格

全站采用「编年历」编辑排版风格（源自 MYSAO 的 `.dc.html` 简报模板）：暖宣纸底
（`#EAE2D3`）+ 浓咖啡 hero/数据条（`#241C12`）+ 金（`#B8924A`）为主强调、青
（`#2E7D55`）与朱（`#B0463A`）作点缀；字体用 Noto Serif SC（衬线标题/卦名）、
Noto Sans SC（正文）、Space Mono（干支/卦名/世应等术数数据与大写标签）。设计
token 集中在 `src/App.jsx` 顶部的 `T` 对象与 `DESIGN_CSS`。

## 部署到 Vercel（推荐，最简单）

### 1. 准备 API key
去 https://console.anthropic.com 注册账号，在 API Keys 页面生成一个 key（形如 `sk-ant-...`），
并在 Billing 页面绑定支付方式（部署后每次调用都会按实际 token 用量扣费，不再是免费的）。

### 2. 本地测试（可选）
```bash
npm install
cp .env.example .env
# 编辑 .env，把 ANTHROPIC_API_KEY 换成你自己的 key
npm install -g vercel
vercel dev
```
浏览器打开 `http://localhost:3000` 测试。

### 3. 正式部署
```bash
npm install -g vercel   # 如果还没装
vercel login            # 用邮箱或GitHub登录
vercel                  # 在项目根目录执行，按提示一路回车即可完成首次部署
```
首次部署后，去 Vercel 后台（vercel.com/dashboard）：
1. 进入这个项目 → Settings → Environment Variables
2. 添加 `ANTHROPIC_API_KEY`，值填你的 key
3. 回到 Deployments，点最新一条的 "Redeploy"（让环境变量生效）

完成后 Vercel 会给你一个 `https://你的项目名.vercel.app` 的网址，就是正式可访问的网站。

### 4. 之后要更新代码
改完代码后，在项目目录里再跑一次：
```bash
vercel --prod
```
即可发布新版本。

## 关于成本
- Claude Sonnet 目前价格大致是每百万输入token几美元、每百万输出token十几美元（具体以 console.anthropic.com 上的定价页为准，会随时间调整）。
- 这个demo每次起局的 prompt 不长，max_tokens=1000，单次请求成本很低，但如果网站被公开访问、流量变大，建议加一个简单的每日调用次数限制（比如按IP或按用户），避免被刷。
- 如果不想承担公开网站的调用成本，也可以先只把链接发给认识的人小范围用。

## 起局/排盘的确定性计算

本站的核心做法是：**起局、排盘、成卦全部由前端确定性算法精确计算，AI 只负责文字解读**（prompt 里明确要求 AI 不得重新起局/排盘/起卦），因此同一局面复算结果稳定。目前已算法计算的部分：

- **八字**：年、月、日、时四柱干支（年/月柱按节气边界判断）。
- **小六壬**：按农历月、日、时辰三步定「大安…空亡」六宫。
- **奇门遁甲**：局数（节气+符头查表）、地盘（三奇六仪入九宫）、值符星、值使门（旬首遁六仪+随时干飞宫/随时辰走门）。
- **梅花易数**：由数字或时间起卦，算出上/下卦、动爻、本卦、互卦、变卦、体用（附五行）。
- **六爻**：模拟三枚铜钱摇六次，算出本卦、变卦、动爻，并按京房八宫定世应。
- **塔罗**：从 22 张大阿尔卡纳随机抽牌（正/逆位随机），按牌阵给出牌位并附正逆位关键词。

## 已知限制
- 奇门遁甲：局数、地盘、值符值使已精确计算；其余七星七门按固定序推演的部分仍由 AI 在解读时补齐（理论上确定性，但未在前端单独展示）。
- 八字的年柱/月柱按节气边界的近似日期表判断，边界日（节气交接前后1天内）出生的可能有偏差，建议用万年历核对。
- 小六壬需要你自己核对准确的农历月/日再输入。
- 梅花易数的**时间起卦**因未接入农历转换库，采用「阳历干支变体」（年支+节气月+阳历日+时辰）；若需严格古法，请改用**数字起卦**。
